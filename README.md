# aacgg

## Development

### Start Project

```sh
yarn install

npm run dev
```

### Start Server

See: [aacgg-server Development](https://github.com/zjffun/aacgg-server/blob/main/README.md#development)

### Set Proxy

- Proxy `https://dev.aacgg.com` to `https://127.0.0.1:3002`
- Proxy `https://dev-api.aacgg.com` to `https://127.0.0.1:30002`

### Open Website

Open <https://dev.aacgg.com/> in browser.
